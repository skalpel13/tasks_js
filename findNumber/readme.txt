Guess the number game. In this game, the computer guesses a number from 1 to 100. 
In the input on the screen, the player enters a number from 1 to 100, trying to guess what the computer guessed.

If the player entered a number less than the guessed one, the computer should write 'Enter a bigger number', 
and if the player entered more than the guessed one, then, accordingly, the computer should write 'Enter a smaller number'.
